# glabq

Clone GitLab repositories using fzf and ghq.

## Requirements

- jq
- [fzf](https://github.com/junegunn/fzf)
- [ghq](https://github.com/x-motemen/ghq)

## Usage

```sh
glabq user arrow2nd
glabq group gitlab-org

glabq -h
```
